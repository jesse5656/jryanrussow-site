# Dictation Inbox

Version: 0.1.0

Status:
Active Capture Area

## Purpose

This folder stores raw dictated narratives captured with Vocalinux.

These files are not polished writing.

They are memory preservation artifacts.

## Rule

Capture before compression.

Do not summarize too early.

Do not force the story to become a lesson immediately.

Preserve the memory first.

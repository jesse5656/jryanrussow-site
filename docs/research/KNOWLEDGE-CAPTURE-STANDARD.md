# Knowledge Capture Standard

Version: 0.1.0

Purpose

The purpose of this standard is to convert experience into transferable
knowledge before it is forgotten.

------------------------------------------------------------------------------

Capture Trigger

Whenever an experience produces:

- a lesson
- a failure
- a success
- a repeated pattern
- an unexpected outcome
- a better method

it should be documented.

------------------------------------------------------------------------------

Knowledge Capture Template

Situation

Observation

What happened?

Why did it happen?

What principle does this suggest?

Can it be generalized?

Can it become a framework?

Can it improve an existing framework?

Does it contradict an existing belief?

Should further research be performed?

------------------------------------------------------------------------------

Goal

Convert tacit knowledge into transferable knowledge.

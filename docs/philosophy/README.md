# The Russow System

This directory contains the intellectual property, philosophy,
frameworks, and long-term research behind everything built by
J. Ryan Russow.

Nothing in this directory is website documentation.

It is the source code for the ideas.

The purpose of these documents is to preserve, improve,
and version the Russow System over decades.

Every website, business, AI system, software product,
book, course, keynote, coaching program, and future venture
should ultimately trace back to these documents.

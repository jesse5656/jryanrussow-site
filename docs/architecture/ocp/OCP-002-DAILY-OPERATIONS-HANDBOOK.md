# OCP-002 — Daily Operations Handbook

Version: 1.0.0

Status:
Approved

Type:
Operational Change Proposal

------------------------------------------------------------------------------

## Purpose

Move the primary day-to-day researcher handbook into the existing researcher operations namespace and rename it Daily Operations.

------------------------------------------------------------------------------

## Current State

The primary operational handbook was stored at:

`docs/handbook/RESEARCHER-HANDBOOK.md`

Its location and title made it difficult to find among constitutional, governance, and institutional handbook files.

------------------------------------------------------------------------------

## Approved Operational Change

Move and rename the handbook to:

`docs/operations/researcher/DAILY-OPERATIONS.md`

The new document is the primary reference for recurring commands, aliases, session procedures, repository-change workflow, and everyday work instructions.

`START-HERE.md` shall point directly to it.

Existing references shall be updated rather than duplicated.

------------------------------------------------------------------------------

## Scope

- `START-HERE.md`
- `docs/handbook/RESEARCHER-HANDBOOK.md`
- `docs/operations/researcher/`
- `docs/architecture/ocp/OCP-002-DAILY-OPERATIONS-HANDBOOK.md`

------------------------------------------------------------------------------

## Governance Boundary

This is an operational documentation relocation and usability improvement.

It does not change constitutional principles, governance authority, or repository architecture outside the existing operations namespace.

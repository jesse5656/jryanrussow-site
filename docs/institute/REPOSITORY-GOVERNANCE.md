# Repository Governance

Version: 0.1.0

Purpose

The repository is the canonical source of truth for the Russow Institute.

No published book, website, course, AI model, presentation, or software
should contradict the repository without updating it first.

------------------------------------------------------------------------------

Repository Hierarchy

1. Constitution

2. Architectural Principles

3. Philosophy

4. Definitions

5. Axioms

6. Frameworks

7. Research

8. Case Studies

9. Publications

------------------------------------------------------------------------------

Every major idea must include:

Definition

Purpose

Research

Evidence

Implementation

Revision History

Open Questions

------------------------------------------------------------------------------

Nothing is ever considered finished.

Only the current best understanding.

# The Russow Institute

Version: 0.1.0

## Purpose

The Russow Institute exists to preserve, develop, test, teach, and improve
the Systems Architect Philosophy.

This repository is not merely website documentation.

It is the source repository for the intellectual property, research,
frameworks, books, software concepts, and educational materials that will
evolve throughout the author's lifetime and continue beyond it.

------------------------------------------------------------------------------

## Mission

Build systems that preserve what matters.

Create frameworks that return time.

Develop people who become better stewards of themselves, their families,
their businesses, and future generations.

## Core Philosophy

Knowledge should always be traceable.

Ideas should never appear without documented origin.

## Guiding Question

Can every important idea be traced back to the observations that produced it?

If not, the work is incomplete.

Midwest24 Archive should eventually model this lineage directly.

Documents become nodes.

Relationships become first-class knowledge.

Institutional memory becomes navigable.

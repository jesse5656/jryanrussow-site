# Primary Source Standard

Version: 1.0.0

Status:
Active Institute Standard

------------------------------------------------------------------------------

## Purpose

Primary Sources preserve reality before interpretation.

They represent the highest-fidelity record available.

------------------------------------------------------------------------------

## Examples

- Vocalinux transcripts
- Journal entries
- Interviews
- Meeting notes
- Emails
- Letters
- Audio recordings
- Video recordings
- Photographs
- Original documents

------------------------------------------------------------------------------

## Capture Rule

Capture before compression.

Preserve before judging.

Interpret later.

------------------------------------------------------------------------------

## Editing Rule

Primary Sources should never be rewritten.

Corrections should be additive.

The original observation remains preserved.

------------------------------------------------------------------------------

## Metadata

Every Primary Source should eventually contain:

- Source ID
- Date Recorded
- Event Date
- Recording Method
- Related People
- Related Organizations
- Related Locations
- Related Research Programs
- Related Field Laboratories
- Related Papers
- Related Canon Candidates

------------------------------------------------------------------------------

## Principle

Experience is not evidence.

Experience becomes evidence through documentation, comparison, analysis, and validation.

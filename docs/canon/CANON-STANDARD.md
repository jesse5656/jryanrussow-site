# Canon Standard

Version: 1.0.0

Status:
Foundational Standard

------------------------------------------------------------------------------

Purpose

The Canon exists to preserve the current best understanding of the
Systems Architect Discipline.

Every chapter included in the Canon should satisfy this standard.

------------------------------------------------------------------------------

Every Chapter Must Answer

What problem exists?

Why does it exist?

Why does it matter?

What principle explains it?

How should someone think differently?

How should someone act differently?

What evidence supports the claims?

What remains uncertain?

------------------------------------------------------------------------------

Every Chapter Must Include

Opening Story

Observation

Definitions

Principles

Framework

Implementation

Reflection Questions

Evidence Map

Revision History

------------------------------------------------------------------------------

Every Chapter Must Be

Understandable

Transferable

Testable

Actionable

Open to Revision

------------------------------------------------------------------------------

Nothing enters the Canon because it sounds good.

Everything enters because it has earned its place.

# Internal Linking Standards

Every article should contain:

1 Link → About

1 Link → Three-Pillar Method

1 Link → Body Code

1 Link → Mind Forge

1 Link → Compound Circle

1 Link → Blog

1 Link → Contact

2–5 Links → Related Articles

------------------------------------------------------------

Never publish an orphan page.

Every page should receive links from at least three other pages.

Cornerstone pages should receive links from nearly every article.

# SEO Page Checklist

Before publishing any page:

- Unique title tag
- Unique meta description
- Canonical URL
- Open Graph title, description, image
- Twitter card metadata
- Article / WebPage schema
- Breadcrumb schema
- FAQ schema when relevant
- Author attribution
- Published and modified dates
- Internal links to About, Method, Body Code, Mind Forge, Compound Circle
- Back link to Blog index where applicable
- Mobile check
- Sitemap updated
